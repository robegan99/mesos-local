Get started with Mesosphere

Welcome to Mesosphere DCOS. Let's maximize your Datacenter resources.

1

Copy the /mesos-usb folder from USB to your computer 

cp -R mesos-usb ~/workspace/mesos-usb

2

Start the Virtual Machine

You'll need Vagrant and VirtualBox installed. 

cd ~/workspace/mesos-usb
vagrant up

3

Access local consoles.

Access Mesos Console

Mesos is the distributed systems kernel at the core of the DCOS. It enables an operating system that spans the entire datacenter.

Launch Mesos Console
Access Marathon Console

Marathon is like a built-in PaaS. Use it to deploy long-running services such as web servers, databases and microservices.

Launch Marathon Console
4

Installation Complete!
